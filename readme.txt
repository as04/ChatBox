|  Real Time Chat with Logs stored Permanently with Username  |
 -------------------------------------------------------------



Pre-Requirements :
----------------

1. LAMP/WAMP/XAMPP server.

2. Node.js.

3. Socket IO.

4. MongoDB.

5. npm.

6. Express.



Installation :
-------------

1. Install LAMP/WAMP/XAMPP.

2. Import the project to respective server location.

3. Run LAMP/WAMP/XAMPP server.

4. Run server.js (inside "Chat/").

5. Run MongoDB.

6. Open index.html from localhost.
